## Project Overview

**HTML-CSS Website** is a fully static web project built to demonstrate **HTML and CSS capabilities** for modern, responsive design. The website features multiple pages including **portfolio, photography galleries, services, team, contact, and blog pages**, with a focus on layout, styling, and image presentation. This project is ideal for showcasing web design skills and structuring content in an organized, visually appealing way.

---

## Features

- Multiple pages for different sections (Home, About, Services, Portfolio, Blog, Contact, Team)  
- Responsive layout for desktop, tablet, and mobile  
- Creative photography galleries with image assets  
- Clean and consistent styling using CSS  
- Demonstrates HTML structure, semantic tags, and static page linking  
- GitHub Pages deployment for live preview
